Project

Security System using RFID and Laser Light

This project is inspired by modern military-grade security systems, designed to enhance safety by preventing unauthorized access and ensuring continuous monitoring of restricted areas.

There was 4 team member including me , i assigned role as a group leader for these projectis Developed as diploma final year project in Electronics and Telecommunication (E&TC), the system integrates RFID authentication with laser-based intrusion detection to provide a smart and automated security solution.

It demonstrates the practical implementation of embedded systems, sensors, and automation techniques to solve real-world security challenges.

The RFID module allows only authorized personnel to access restricted areas using RFID cards. Unauthorized access attempts are blocked instantly.

The laser system acts as a virtual barrier. When interrupted, it triggers an alarm for real-time intrusion detection.

This project improved my skills in circuit design, sensor integration, and microcontroller programming.
